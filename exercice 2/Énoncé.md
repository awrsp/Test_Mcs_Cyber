# Exercice 03 : Docker

**Création d’un fichier docker-compose pour gérer un site Wordpress :**
A partir des images nginx, mariadb, php, installez la dernière version de Wordpress

**Vous devez utiliser un volume commun pour les data / configs**

**Mettre dans votre Github :**
- un fichier Readme.md pour expliquer votre process 
- votre fichier docker-compose.yaml 
- votre fichier de configuration nginx
- tout autre fichier nécessaire
